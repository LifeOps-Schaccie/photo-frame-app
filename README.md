# 📷 Photo Frame App

写真にマット（台紙）と撮影情報を合成してダウンロードできる、ブラウザで動作する静的Webアプリです。

**👉 [アプリを開く](https://lifeops-schaccie.github.io/photo-frame-app/)**

---

## 特徴

- **インストール不要** — ブラウザだけで動作。サーバーへのアップロードなし
- **プライバシー安全** — 写真はすべてブラウザ内で処理され、外部に送信されません
- **EXIF自動取得** — カメラ・レンズ・F値・シャッタースピード・ISO・撮影日を自動読み込み
- **複数枚対応** — まとめて読み込み、一括ダウンロード可能

## 機能

| 機能         | 内容                                                                           |
| ------------ | ------------------------------------------------------------------------------ |
| 写真読み込み | ドラッグ＆ドロップ または クリック選択（複数可）                               |
| スタイル     | 台紙色・文字色のプリセット4種 + カラーピッカーで自由設定                       |
| フォント     | Noto Sans JP / Noto Serif JP / Inter                                           |
| アスペクト比 | Auto / 4:5 / 5:4 / 3:2 / 2:3                                                   |
| 余白         | 2〜10%（スライダー）                                                           |
| 表示情報     | カメラ/レンズ・設定値・撮影日・作者・撮影場所（各行 表示ON/OFF・太字切り替え） |
| エクスポート | PNG / JPEG、最大辺 2000〜4000px または 元サイズ                                |

## 使い方

1. [アプリを開く](https://lifeops-schaccie.github.io/photo-frame-app/)
2. 写真をドラッグ＆ドロップ（または「写真を選択」をクリック）
3. 右パネルでスタイルと表示情報を調整
4. 「ダウンロード」または「すべてダウンロード」をクリック

## 技術スタック

- HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- [exifr](https://github.com/MikeKovarik/exifr) — EXIF読み取り（MIT License）
